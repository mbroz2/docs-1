:linkcss:

// tag::description[]
This feature provides support for the Eclipse MicroProfile Metrics 2.2 specification for enterprise Java.

// end::description[]
// tag::enable[]
== Enabling this feature
To enable the MicroProfile Metrics 2.2 feature, add the following element declaration into your `server.xml` file, inside the `featureManager` element:


----
<feature>mpMetrics-2.2</feature>
----
// end::enable[]
